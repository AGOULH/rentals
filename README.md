# نظام إدارة الإيجارات

تطبيق ويب من ملف واحد (`index.html`) لإدارة العقارات ومتابعة دفعات الإيجار الربع سنوية — بلا اعتماديات ويعمل دون إنترنت.

## المزايا
- إدارة العقارات والوحدات (شقق/محلات) تحت كل عقار.
- جدول دفعات ربع سنوي تلقائي:
  `تاريخ الدفعة (ن) = تاريخ بداية العقد + (ن−1) × 3 أشهر`
- لوحة تحكم: متأخرات، دفعات قريبة الاستحقاق، عقارات قريبة التسليم، عقود قريبة الانتهاء.
- تسجيل السداد وتتبّع المحصّل والمتبقّي لكل وحدة.
- تصدير PDF لكل وحدة بكشف الدفعات وتواريخها.
- نسخ احتياطي/استيراد للبيانات بصيغة JSON.

## التشغيل
افتح `index.html` في المتصفح مباشرةً. تُحفظ البيانات محليًا في المتصفح عبر `localStorage` (المفتاح `rentals_app_v1`) ولا تغادر جهازك.
